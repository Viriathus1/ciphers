# Ciphers

A really simple and frankly quite rough around the edges GUI that helped me tackle the
[cipher challenges](https://simonsingh.net/cryptography/cipher-challenge/) found in
[The Code Book](https://en.wikipedia.org/wiki/The_Code_Book) by Simon Singh.

Read more about it [here](https://viriathus.dev/cipher-challenges/).

## How to Install

```bash
go install github.com/Viriathus1/ciphers@latest
```
